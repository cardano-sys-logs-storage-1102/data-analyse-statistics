03 — resultados

análise por categoria

como interpretar a Análise por Categoria

Esta seção compara subpopulações dos seus dados. Analise se as Médias e Medianas variam significativamente entre os grupos, o que indica que a categoria influencia o resultado numérico.

Verifique também o Desvio Padrão (DP): um grupo com DP muito maior possui maior volatilidade e é menos previsível que os demais.

como ler este gráfico

O histograma divide os valores em intervalos (bins) e mostra a frequência de cada um. A curva sobreposta é a KDE (Estimativa de Densidade de Kernel), uma versão suavizada da distribuição. No eixo X estão os valores da variável; no eixo Y, a contagem de ocorrências em cada bin. As linhas verticais marcam a média (laranja tracejado), a mediana (ciano sólido) e a moda (violeta pontilhado).

como ler este gráfico

O box plot resume a distribuição em cinco números: mínimo, Q1 (25%), mediana (50%), Q3 (75%) e máximo. O retângulo (caixa) vai de Q1 a Q3 e representa o IQR. A linha dentro da caixa é a mediana. As whiskers (hastes) se estendem até os valores mais extremos dentro de 1,5× o IQR a partir da caixa. Pontos fora das whiskers são outliers (em vermelho).

como ler este gráfico

O Q-Q Plot (Gráfico Quantil-Quantil) compara os quantis observados da amostra com os quantis teóricos de uma distribuição normal. Quando os pontos seguem a linha diagonal de referência (μ + σ·x), os dados têm distribuição aproximadamente normal. Desvios sistemáticos indicam assimetria ou caudas pesadas. A banda de confiança a 95% (Kolmogorov-Smirnov) delimita a região de normalidade esperada.

como ler este gráfico

O violin plot combina um box plot com a estimativa de densidade KDE (kernel gaussiano, bandwidth de Silverman). A largura do violin em cada altura representa a densidade dos dados naquele valor. O retângulo central mostra o IQR (Q1–Q3), a linha branca é a mediana e o losango teal é a média. Para grupos com menos de 10 pontos, os dados individuais são exibidos com jitter horizontal.

como ler este gráfico

O scatter plot exibe cada valor como um ponto, com o índice da observação no eixo X e o valor no eixo Y. Pontos azuis são valores normais (dentro de 1,5× IQR). Pontos vermelhos são outliers (fora dos limites de Tukey). A linha tracejada laranja indica a média; as linhas verdes delimitam a faixa de ±2 desvios padrão.

como ler este gráfico

O gráfico de barras por grupo compara a média da variável numérica em cada categoria. Cada barra representa um grupo; sua altura é a média do grupo. Passe o mouse sobre uma barra para ver o desvio padrão (σ) do grupo, que indica a dispersão interna.

como ler este gráfico

A ECDF (Função de Distribuição Empírica Acumulada) mostra, para cada valor no eixo X, qual proporção dos dados é menor ou igual a ele. O eixo Y vai de 0% a 100%. A banda de Greenwood (95%) mostra a incerteza estatística da estimativa. As linhas horizontais tracejadas marcam os quartis Q1, Q2 e Q3.
