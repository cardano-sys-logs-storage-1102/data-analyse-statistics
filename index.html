moda · gini · entropia
o que é gini e entropia normalizada?

Coeficiente de Gini: mede a concentração da distribuição. 0 = perfeitamente uniforme (todos os valores aparecem igualmente), 1 = totalmente concentrado (um valor domina). Valores próximos de 0.7+ indicam alta concentração.

Entropia normalizada (0–1): quão "diversa" é a distribuição. 1 = máxima diversidade, 0 = um único valor. É o complemento perceptivo do Gini.

// 02 · descritiva

Estatísticas Descritivas

Medidas de tendência central, dispersão, forma e posição para uma coluna numérica.
✦
Insights automáticos
como interpretar estes valores

Média vs mediana: se forem muito diferentes, há assimetria ou outliers. A mediana é mais robusta.

Desvio padrão (σ): em média, o quanto os dados se afastam da média. ~68% dos valores caem em [μ−σ, μ+σ] se for normal.

IQR: intervalo onde estão os 50% centrais dos dados (Q1 até Q3). Base para detectar outliers.

CV%: dispersão relativa. <15% homogêneo, 15–30% moderado, >30% alto.

Assimetria (skewness): 0 = simétrico, >0 = cauda à direita, <0 = à esquerda.

Curtose: quão pesadas são as caudas vs. normal. 0 = normal, >0 = leptocúrtica (caudas pesadas).

como interpretar Q-Q plot e probabilidades

Q-Q plot: compara quantis dos seus dados (eixo Y) com quantis teóricos da distribuição selecionada (eixo X). Se os pontos se alinham na diagonal, a distribuição é uma boa aproximação.

Desvios na cauda direita sugerem cauda mais pesada que o esperado; na esquerda, dados mais concentrados próximos à média.

Inverso (quantil): dado uma probabilidade P, retorna o valor x tal que P(X ≤ x) = P. Útil para definir limites (ex: percentil 95%).

Discretas (Poisson, Binomial): usam P(X = k) ao invés de PDF — exibimos como barras.

// 04 · testes

Testes Estatísticos

Normalidade, comparação de grupos e independência entre variáveis categóricas.
N
Teste de Normalidade
Shapiro-Wilk
como interpretar estes resultados

H₀ (hipótese nula): os dados são normais. H₁: não são.

Se p > 0.05 → não rejeitamos H₀ → compatível com normal.

Se p ≤ 0.05 → rejeitamos H₀ → não é normal. Use testes não-paramétricos (Kruskal-Wallis, Mann-Whitney etc.).

A
ANOVA / Kruskal-Wallis
comparar grupos
como interpretar estes resultados

ANOVA: teste paramétrico — requer normalidade e homogeneidade de variâncias.

Kruskal-Wallis: versão não-paramétrica (ranks), robusto a desvios de normalidade.

Se p < 0.05 em algum deles, há diferença significativa entre pelo menos dois grupos.

χ
Qui-Quadrado · Independência
2 categóricas
como interpretar estes resultados

H₀: as duas variáveis são independentes.

Se p < 0.05, existe associação estatisticamente significativa.

V de Cramér: mede a força da associação. 0 = nenhuma, 1 = perfeita. >0.3 é considerado relevante.

// 06 · comparação

Comparar 2 Colunas

Métricas lado a lado, diferença de médias e correlação de Pearson.
⇌
Comparação entre colunas
como interpretar estes resultados

Δμ: quanto a média da coluna B difere da coluna A (em valor absoluto e %).

r de Pearson: correlação linear. |r| < 0.2 muito fraca, 0.2–0.4 fraca, 0.4–0.6 moderada, 0.6–0.8 forte, >0.8 muito forte.

CV%: coluna com CV maior é mais dispersa em relação à sua média.
